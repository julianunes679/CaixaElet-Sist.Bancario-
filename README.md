Caixa Eletronico - Sistema Bancário em Python

Projeto didático desenvolvido para praticar Python, lógica de programação e Programação Orientada a Objetos (POO).
O sistema simula operações bancárias reais, como criar contas, fazer login, depositar, sacar e consultar saldo.

Funcionalidades
    Criar contas com número gerado automaticamente
    Login por número da conta
    Ver saldo
    Fazer depósitos
    Realizar saques
    Menu interativo dentro da conta
    Encapsulamento com atributos privados (__saldo)
    Código organizado em módulos
